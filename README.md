# Products App

SPA-приложение со списком карточек товаров, созданное в рамках тестового задания на позицию Frontend Developer (React / Next.js).

## Технологии

- Next.js
- React
- TypeScript
- Redux Toolkit
- Tailwind CSS

## Функционал

### Список товаров `/products`

- загрузка данных с публичного API Dog API
- отображение карточек товаров
- изображение, название, описание, категория
- лайк товара
- удаление карточки
- фильтр:
  - Все
  - Избранные
- переход на страницу товара

### Страница товара `/products/[id]`

- подробная информация о товаре
- изображение
- описание
- категория
- кнопка возврата к списку

### Создание товара `/create-product`

- форма создания карточки
- обязательные поля
- минимальная валидация
- сохранение в Redux store
- редирект на страницу товаров после создания

### Дополнительно

- сохранение данных в localStorage
- навигация между страницами
- адаптивная сетка карточек
